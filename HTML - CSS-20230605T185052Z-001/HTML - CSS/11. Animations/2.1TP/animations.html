ANIMATIONS
Les transitions permettent de faire des animations simples en passant d’un état A à un état B Pour faire cela, nous allons utiliser deux propriétés: transition-property => Permet de choisir quelle(s) propriété(s) CSS va être animée transition-duration => Permet de choisir le temps sur lequel se déroule l’animation
